# Server Stats

A simple Bash script that analyzes basic Linux server performance statistics.

## Features

- Total CPU usage
- Total memory usage
- Total disk usage
- Top 5 processes by CPU usage
- Top 5 processes by memory usage

Go to the project directory:

```bash
cd server-status
```

Make the script executable:

```bash
chmod +x server-status.sh
```

## Usage

Run the script:

```bash
./server-status.sh
```


## Technologies Used

- Bash
- Linux command-line utilities (`top`, `free`, `df`, `ps`, `awk`)
